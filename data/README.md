# Data Directory

This directory contains all data files used by the TRIDENT system.

## Structure

- `raw/` - Original, unprocessed data
  - `weather/` - Historical weather data
  - `reference_datasets/` - External reference datasets
- `models/` - Trained machine learning models
- `visualizations/` - Generated charts and graphs
- `databases/` - SQLite databases

## Usage

- Raw datasets are used for training ML models
- Models directory contains trained model artifacts
- Visualizations are generated by the ML pipeline
- Databases store operational data

## Data Sources

- Chennai historical weather data
- Emergency response records
- Sensor calibration data
- Reference datasets for testing
